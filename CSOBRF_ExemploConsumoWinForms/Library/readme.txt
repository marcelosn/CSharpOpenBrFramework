Pra rodar o projeto:
Antes de abrir a solução, crie uma pasta no C:\CSharpOpenBrFramework. Após isso, copie o conteúdo da pasta "Librarys" (dentro do exemplo WinForms) para esse diretório.
Pronto, só abrir e compilar. Não se esqueça de setar o projeto de Exemplos (WinForms) como principal para rodar.
